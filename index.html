Vision-Based Models for Sorting and Segregation of Waste

[ Code ]   [ Slides ]  [ Demo ]  

This work was done during an internship at Omdena. We built a object classifcation and detection based models to segregate trash into various common categories.We created a new dataset containing over 16000 images of various trash items. We tested various architectures Deep Nets and SSDs and achieved an overall accuracy of 96.14% for 10-class classification of trash.Finally proposed and presented the overall solution as an UN-SDG India team as a proof-of-concept.
Tech Saplings: Intelligent Environment and Plant Health Monitoring and Evaluation System

[ Code ]   [ Slides ]  

We propose an end-to-end system to monitor the environmental conditions and health of the plant by using variety of embedded sensors to measure conductivity, pH levels, soil moisture etc. in addition to a drone-based camera system to counting and imaging purposes for data collection. For the purpose of predicting diseases, drooping, fire and for health condition evaluation etc. this data is input into object detection and segmentation models as well as prediction models.We found that this low cost system is an effective solution for inspection of forests and other plantations intiatives.This project won the 1st prize at the Smart Odisha Hackathon-2022.
Pancreatic 3D-Segmentation using CT Scans 🔗

In this work, we built a transformer architecture for 3D medical image segmentation using an encoder-decoder style architecture with linear complexity based on an exisiting paper.We evaluate this method on various CT pancreas segmentation datasets for cancer detection.We achieved an overall accuracy of 91.32% and leveraging a lower number of annotations to provide highly feasible segmentation performance as well as accurate uncertainty quantification.

Satellite Data based Pollution Forecasting using CNNs 🔗

This work investigates the feasibility and accuracy of using a convolutional ceural cetwork upon satellite images to predict the Breezometer Air Quality Index (BAQI). We propose a new BAQI index dataset with over 10000 satellite images and respective Breezometer air quality data records across 57 cities. We tested various architectures deep CNN models with variations in layers,data augumentation techniques and achieved an overall accuracy of 87.14%. We find that satellite images, which are inexpensive and ubiquitous, are accurate in predicting air pollution.

CoughVid: Covid-19 Detection from Cough Voice Samples 🔗

This work was done for the Pfizer Digital Medicine Challenge.We built a RNN and MFCC feature extraction based model to detect Covid-19 from coughing voice Samples.We achieved an overall accuracy of 82.42% in real day testing on over 320 patients. We find that MFCC and Spectrograms of the audio signals can also be used as image dataset and build CNN Models to classify the audio samples
